DATA-DRIVEN-STOCK-ANALYSIS

Project Overview:
This project is a comprehensive Stock Market Analytics Dashboard built using Python, SQLite, and Streamlit.
It performs end-to-end financial data analysis, including volatility analysis, sector performance, cumulative returns, correlation analysis, and market-level insights.

Project Objectives:
Analyze historical stock market data, Identify top-performing and underperforming stocks, Visualize sector-wise and time-based trends, Provide an interactive dashboard for data exploration, Store and manage data efficiently using SQLite

Tech Stack:
1.Language	Python
2.Database	SQLite
3.Visualization	Matplotlib, Seaborn, Streamlit, Power BI
4.Data Processing	Pandas, NumPy

Features Implemented:

✔ Market Overview
Total number of stocks
Average stock price
Average  volume
Count of gainers vs losers

✔ Volatility Analysis
Top 10 most volatile stocks
Visual bar charts

✔ Sector-wise Performance
Sector-level average returns
Comparative visualization

✔ Cumulative Returns
Cumulative return

✔ Monthly Gainers & Losers
Top 5 monthly gainers & losers
Bar chart comparisons

Heatmap visualization of stock correlations
